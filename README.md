# Trius Lounge Book List
광명 트리우스 아파트 내 라운지에 구비된 도서를 검색할 수 있습니다. 

## 1. 기획 의도
라운지에 수천 권이 책이 들어왔지만, 어떤 책이 있는지 미리 검색할 수가 없어서 활용을 잘 못하고 있었습니다. 이에 AI 로 프로그래밍을 배우기 시작하면서 첫 토이 프로젝트로 라운지의 책을 검색할 수 있는 페이지를 만들었습니다. 

## 2. 기능
* 도서 검색: 키워드로 책을 검색할 수 있습니다.
* 추천 대상: 도서 내용을 기반으로 누구에게 도움이 되는 책인지를 알 수 있습니다.
* 도서 정보 프린트: 원하는 책의 상세정보와 책번호 등을 확인하고 프린트할 수 있습니다. 모바일로 상세정보를 캡처한 후 라운지를 방문하면, 원하는 책을 보다 쉽게 찾을 수 있습니다.
  
## 3. 전략적 접근 방식
비개발자로서 쉽게 업데이트를 할 수 있고 효율적으로 관리할 수 있는 데 집중했습니다.

* 데이터 구조의 간결함: 복잡한 데이터베이스 대신 CSV 형식을 선택하여 관리자가 엑셀이나 메모장만으로도 리스트를 즉시 업데이트할 수 있습니다.
* 민첩한 배포: GitHub Pages를 활용하여 별도의 서버 비용 없이 정적 사이트를 호스팅함으로써 유지보수 비용을 최소화했습니다.
* 확장성: 현재는 도서 목록에 집중하고 있으나, 향후 각 도서에 대한 리뷰나 방문자들의 코멘트를 수집할 수도 있습니다.

## 4. 기술 스택 (Tech Stack)
데이터 처리의 자동화와 웹 시각화를 위해 아래의 도구들을 활용했습니다.
* Frontend: HTML5, CSS3
* Backend: Python, Flask
* Infrastructure: GitHub Actions & Pages
